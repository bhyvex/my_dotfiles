Installation
============

Use solarized dircolors from:
https://github.com/seebi/dircolors-solarized/blob/master/dircolors.256dark

```sh
git clone --recursive https://github.com/darnuria/my_dotfiles
cd my_dotfiles
bash install.sh
```
